Soft Django Rest Framework

Inicio Instalación Uso

Soft Django Rest Framework es un conjunto de mixins y templates para hacer scaffolding con Django Rest Framework.

Instalación

  1. pip install soft_drf

  2. En tus INSTALLED_APPS agrega:
    INSTALLED_APPS = [
         'soft_drf',
    ]

Uso

  1. Edit settings.py, and then run your project.

  2. Open your browser in http://localhost:8000/api/v1

# soft_drf
Mixins and templates to do scaffolding with Django Rest Framework project.

## Install
```json
    pip install soft_drf
```


## Requirements
* Python (3.5, 3.6)
* Django (1.10, 1.11)
* Django REST Framework (3.5, 3.6)
